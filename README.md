# Food Ordering App - Frontend

A modern Food Ordering Application frontend built using React.js. The application allows users to browse food items, search and filter products, manage a shopping cart, and view dashboard statistics through seamless integration with backend APIs.

## Features

* Food Listing
* Search Functionality
* Category Filtering
* Shopping Cart Management
* Dashboard Statistics
* Responsive User Interface
* REST API Integration

## Tech Stack

* React.js
* JavaScript
* CSS3
* HTML5

## Deployment

```bash
git clone <repository-url>
cd food-ordering-frontend
npm install
npm start
```
